# Voxel Editor
Very simple voxel editor.
- Supports different materials: color, smoothness, metallic, emission, transparency.
- Frustum Culling
- Backface Culling
- Binary Greedy Meshing with hidden faces culling and logic for transparent voxels
- Simple undo/redo system with command pattern
- UI decoupled from backend
